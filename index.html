Big Integer Class

Written in C++

Have you ever used C++ and been irritated by how the biggest
number you could come up with was from an unsigned long long int, which
maxes out at 18,446,744,073,709,551,616? Well then use my C++ Big class! By using
vectors I was able to rewrite basic digit-wise mathematical operators, such that you
you could easily integrate it into your own code!
